primus-mark-visited

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

Marks all terms visited by any Primus machine with the [Term.visited] attribute and terms that weren't visited. with [Term.dead]. Terms will not be marked visited during the execution, but only after a system finishes.
